## Auction
Auctora is a simple web-based auction system built using HTML, CSS, and JavaScript. It allows users to view items, place bids, and see prices converted into Kenyan Shillings (KSh).

## Features
Displays products from a public API
Simple bidding system in the browser
Currency conversion (USD to KSh)
Responsive layout for different screen sizes
Multi-page structure (Home, Auctions, Dashboard)
## Project Structure
index.html
auction.html
dashboard.html
style.css
script.js
## API Used
DummyJSON
## How to Run
Open index.html in a browser
Or use Live Server in VS Code for better experience
## Currency Conversion
All prices are converted from USD to KSh using a fixed rate:
1 USD = 160 KES
## Note
This project is frontend-only and does not store data permanently.
